# 加载中 Loading

Loading 用于显示正在加载中状态

### Loading 类型
weui Loading提供两种类型，default，primary

<weui-loading type="default"></weui-loading>
<weui-loading type="primary"></weui-loading>

::: details 显示代码
```vue
<weui-loading type="default"></weui-loading>
<weui-loading type="primary"></weui-loading>
```
:::

### API
#### Loading Props
|  名称   | 类型  | 默认值 | 说明 | 版本 |
|  ----  | ----  | ----- | ---- | ----- |
| type  | 'default' \| 'primary' | 'default' | Loading类型 | - |
